# 🚀 Get Your Live Link - Step by Step

## ⚡ FASTEST METHOD (Recommended): Vercel Web Interface

### Steps:
1. **Open your browser** and go to: **https://vercel.com/new**
2. **Sign up/Login** (free - use GitHub, Google, or email)
3. **Click "Browse"** or drag and drop your project folder:
   ```
   C:\Users\Ax\OneDrive\Videos\Desktop\FMS_frontend
   ```
4. **Verify settings** (usually auto-detected):
   - Framework: **Vite**
   - Build Command: `npm run build`
   - Output Directory: `dist`
   - Root Directory: `.` (current directory)
5. **Click "Deploy"**
6. **Wait 30-60 seconds**
7. **Your app is live!** Copy the URL (e.g., `https://fms-frontend.vercel.app`)

### ✅ Done! Share your link with anyone!
