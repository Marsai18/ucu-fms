# 🚀 Get Your Live Link - Quick Guide

## ⚡ FASTEST METHOD: Vercel Web Interface (2 minutes)

### Step-by-Step:
1. **Go to Vercel**: https://vercel.com/new
2. **Sign up/Login** (free account)
3. **Click "Browse"** and select your project folder:
   ```
   C:\Users\Ax\OneDrive\Videos\Desktop\FMS_frontend
   ```
4. **Configure** (usually auto-detected):
   - Framework: **Vite**
   - Build Command: `npm run build`
   - Output Directory: `dist`
5. **Click "Deploy"**
6. **Wait 30-60 seconds**
7. **Get your live URL!** (e.g., `https://fms-frontend.vercel.app`)

### That's it! Your app is live! 🎉

## 🔗 Temporary Link: ngrok

If you need a quick temporary link for testing:

```bash
# Terminal 1: Start dev server
npm run dev

# Terminal 2: Create tunnel
ngrok http 3000
```

You'll get a URL like: `https://abc123.ngrok.io`

**Note**: This only works while your dev server is running.
